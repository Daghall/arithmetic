# Commands

| Key | Command |
| --- | --- |
| `0-9`   | Set active **operand** |
| `Up`    | Increase active **operand** |
| `Down`  | Decrease active **operand** |
| `Tab`   | Switch active **operand** |
| `Enter` | Swap **operands** |
| `Space` | Cycle *operation*: `+`, `−`, `×`, `÷` |
| `+`     | Set *operation* to `+`|
| `-`     | Set *operation* to `−`|
| `*`     | Set *operation* to `×`|
| `/`     | Set *operation* to `÷`|
